<!-- 🔵 1xx – Informational (Thông tin)
Yêu cầu đã được nhận và server đang tiếp tục xử lý.
Mã	Ý nghĩa
100	Continue
101	Switching Protocols
102	Processing

🟢 2xx – Success (Thành công)
Yêu cầu được xử lý thành công.

Mã	Ý nghĩa
200	OK
201	Created
202	Accepted
204	No Content (không có dữ liệu trả về)

🟡 3xx – Redirection (Chuyển hướng)
Client cần thực hiện hành động khác để hoàn tất yêu cầu.

Mã	Ý nghĩa
301	Moved Permanently
302	Found (tạm thời)
304	Not Modified

🔴 4xx – Client Error (Lỗi từ phía client)
Yêu cầu sai hoặc không hợp lệ từ phía người dùng.

Mã	Ý nghĩa
400	Bad Request
401	Unauthorized (chưa đăng nhập)
403	Forbidden (không có quyền)
404	Not Found
409	Conflict (xung đột dữ liệu)

🔴 5xx – Server Error (Lỗi từ phía server)
Server gặp lỗi khi xử lý yêu cầu hợp lệ.

Mã	Ý nghĩa
500	Internal Server Error
501	Not Implemented
502	Bad Gateway
503	Service Unavailable
504	Gateway Timeout -->
